Hướng dẫn chạy app trên android studio :
B1 : Cần có android studio và cài các bản giả lập điện thoại
B2 : Khởi động giả lập điện thoại trong android studio (chờ mở lên )
B3 : Trong Terminal của folder project chạy lệnh npm start ... (wait a second) rồi bấm nút a để tự mở app trên android studio
B4 : Thao tác ....
